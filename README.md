Computer Networks Lab - README
About

This repository contains Computer Networks Lab programs for students. The programs are written in Java and cover basic networking concepts that are commonly included in undergraduate laboratory courses.

Objectives
Understand the fundamentals of computer networks.
Learn different networking algorithms and protocols.
Implement networking concepts using Java.
Improve programming and problem-solving skills.
Programs Included
Bit Stuffing
Bit De-stuffing
CRC (Cyclic Redundancy Check)
Hamming Code
Checksum
Sliding Window Protocol
Stop-and-Wait Protocol
Go-Back-N ARQ
Selective Repeat ARQ
Distance Vector Routing
Leaky Bucket Algorithm
Token Bucket Algorithm
Requirements
Java JDK 8 or above
Any Java IDE (Eclipse, IntelliJ IDEA, NetBeans) or Command Prompt/Terminal
How to Run
Download or clone the repository.
Open the project in your preferred Java IDE.
Compile the required Java program.
Run the program.
Enter the required input and observe the output.
Learning Outcomes

After completing these lab programs, students will be able to:

Understand data transmission techniques.
Perform error detection and error correction.
Learn flow control and congestion control mechanisms.
Understand routing algorithms and network protocols.
Gain hands-on experience with Java-based networking implementations.
Note

These programs are developed for educational and laboratory purposes. They are designed to help students understand the concepts of computer networks through simple and easy-to-follow implementations.
